# ASOIU_sem4

## Что делает проект

Приложение:
- создаёт набор геометрических фигур;
- вычисляет и выводит их площади;
- размещает фигуры в разреженной матрице;
- хранит фигуры в собственном односвязном списке;
- сортирует список по площади фигур;
- показывает работу стека на основе списка.

## Реализованные сущности

- `Figure` — абстрактная базовая фигура с поддержкой сравнения по площади;
- `Rectangle`, `Square`, `Circle` — конкретные геометрические фигуры;
- `Matrix<T>` — обобщённая разреженная матрица на словаре;
- `IMatrixCheckEmpty<T>` — стратегия определения пустого элемента матрицы;
- `FigureMatrixCheckEmpty` — реализация пустого элемента для фигур;
- `SimpleList<T>` — собственный односвязный список;
- `SimpleStack<T>` — стек поверх `SimpleList<T>`.

## Структура репозитория

- `sem4/` — исходный код консольного приложения;
- `sem4/Program.cs` — демонстрационный сценарий работы приложения;
- `sem4/sem4.csproj` — файл проекта .NET;
- `seminar4.pdf` — сопутствующий материал, лежащий в корне репозитория.

```bash

```

```bash
cd sem4 && dotnet build
```
